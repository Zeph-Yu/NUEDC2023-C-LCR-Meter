# IQ 解调参考信号表生成

> 解释 `IQDemod.c` 中 `IQ_cos_table[40]` 和 `IQ_sin_table[40]` 的计算方法

---

## 参数关系

```
f_s   = 4 MHz      (ADC 采样率)
f_exc = 100 kHz    (激励信号频率)

N = f_s / f_exc = 4,000,000 / 100,000 = 40
```

ADC 在每个激励周期内采 40 个点，一个完整 $2\pi$ 周期被均分为 40 份，每份 $9^\circ$。

---

## 计算公式

$$IQ\_cos\_table[k] = \cos\left(\frac{2\pi k}{N}\right) = \cos\left(\frac{2\pi k}{40}\right)$$

$$IQ\_sin\_table[k] = \sin\left(\frac{2\pi k}{N}\right) = \sin\left(\frac{2\pi k}{40}\right)$$

其中 $k = 0, 1, 2, \dots, 39$。

---

## 生成代码

一行 Python 即可生成完整的 C 数组：

```python
import math

for k in range(40):
    print(f"    {math.cos(2 * math.pi * k / 40):10.6f},  // k={k:2d}  cos({k * 9:3d}°)")

print()

for k in range(40):
    print(f"    {math.sin(2 * math.pi * k / 40):10.6f},  // k={k:2d}  sin({k * 9:3d}°)")
```

## 数值精度说明

表值取 6 位小数（double 精度足够）。验证几个关键点：

| k | 角度 | 理论值 | 表值 | 误差 |
|---|------|--------|------|------|
| 0 | 0° | $\cos 0° = 1$ | 1.000000 | $< 10^{-6}$ |
| 10 | 90° | $\cos 90° = 0$ | 0.000000 | 精确 |
| 20 | 180° | $\cos 180° = -1$ | -1.000000 | $< 10^{-6}$ |
| 5 | 45° | $\cos 45° = \sqrt{2}/2$ | 0.707107 | $< 10^{-6}$ |

12-bit ADC（4096 级，每级约 0.8mV）下 6 位小数的表格精度完全足够，表格量化误差远小于 ADC 量化误差。

## 查找方式

解调时按采样点索引循环查表：

```c
for (int i = 0; i < count; i++)
{
    int k = i % IQ_N;  // i % 40，自动循环回绕

    I_acc += v_ac * IQ_cos_table[k];  // × cos(2πk/40)
    Q_acc += v_ac * IQ_sin_table[k];  // × sin(2πk/40)
}
```

第 0 个采样点用 $k=0$（cos 和 sin 在 $0^\circ$ 的值），
第 40 个采样点回到 $k=0$（下一个周期的起点），以此类推。

## 物理含义

cos 表和 sin 表分别对应锁相放大器的**同相参考**和**正交参考**：

```
ADC 采样序列 v[0], v[1], v[2], ...

v[i] × cos_table[i % 40]  → 同相分量 I（测量与 cos 的相关性）
v[i] × sin_table[i % 40]  → 正交分量 Q（测量与 sin 的相关性）
```

I 分量反映输入信号中 $\cos(\omega t)$ 成分的幅值，Q 分量反映 $\sin(\omega t)$ 成分的幅值。二者构成复数 $I + jQ$，完整描述了 100kHz 频率分量的幅值和相位。
